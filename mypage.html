インターネット分離導入の背景

サイバー攻撃は年々巧妙かつ高度化しており、ランサムウェア等のマルウェアや悪意のある攻撃者による被害が後を絶ちません。
これらの攻撃の多くはインターネットを利用して様々な手法で実行されており、例えばドライブバイダウンロード攻撃のように、いつも訪れているWebサイトを閲覧するだけでマルウェアに感染するなど、インターネットを利用するだけで知らず知らずのうちにマルウェアに感染することもあります。
これら攻撃への対策として企業は様々なセキュリティ対策をしていますが、インターネットに接続されている以上完全に防御することは難しいのが現状です。

インターネット分離とは？(web分離）

この状況を打開する方法の一つとして、インターネットへの接続環境を分離することで、サイバー攻撃のリスクを抑止する『インターネット分離（Web分離）』による防衛手法が注目されています。インターネット分離は、企業の機密情報、個人情報の流出を最小限に抑えるための対策として、独立行政法人情報処理推進機構（IPA）や総務省、文科省、経済産業省等からも、推奨されています。

主なインターネット分離(Web分離)方式について

インターネット分離(Web分離)の最もポピュラーな方法は、インターネット専用のパソコン用意して、社内ネットワークから分離する方法です。しかしこの方法では社内ネットワークとは別にインターネット専用パソコンとそのセキュリティ対策も必要になるためコストが掛かり、パソコンを使い分けるユーザーの負担も大きくなるためあまり普及しませんでした。そこで登場したのがインターネット分離ソリューションです。現在のインターネット分離ソリューションには大きく3つの方式があります

RevoWorksによるインターネット分離のご提案

RevoWorks SCVX と RevoWorks Browser は、インターネット分離を実現するための製品です。 これらの製品は、仮想ブラウザを実行する環境が、サーバー上のコンテナかローカル PC 内のコンテナかという違いはありますが、業務端末とインターネット間の通信を分離するという点では共通しています。

 仮想ブラウザは、 RevoWorks SCVX および RevoWorks Browser の終了時にコンテナごと削除されるため、仮にマルウェアのような不正なファイルをダウンロードしてしまったとしても、業務端末および業務を行うネットワークに侵入されることはありません。 

また、業務端末とコンテナは分離されているため、業務端末内のデータ（重要な情報資産など）がコンテナを介してインターネットに漏えいすることはありません。さらに、インターネットから取得したファイルを業務端末内に持ち込む必要があるケースは多いと思います。しかし、業務端末にマルウェアなど不正なファイルが侵入することは防がなくてはなりません。 

RevoWorks SCVX と RevoWorks Browser は、ウイルスチェックだけでなく、無害化処理やサンドボックスチェックを行ってから業務端末内に安全にダウンロードする仕組みを内蔵しており、これらの処理を行うために別のサーバーやシステムを用意する必要はありません。

システムや運用に求められる要件やユーザビリティを考慮してRevoWorks SCVX とRevoWorks Browser のどちらかを選択してください。

サーバー型仮想ブラウザ

RevoWorks SCVX

RevoWorks SCVX は、 Linux ベースの Docker コンテナ技術によるインターネット分離製品です。

Linuxサーバー上のコンテナに仮想ブラウザを起ち上げ、画面転送技術より、インターネットをセキュアに閲覧することができます。

RevoWorks SCVXの詳細はこちらから　〉

ローカル仮想ブラウザ

RevoWorks Browser

RevoWorks Browserは、自社開発したローカルコンテナ技術によるインターネット分離製品です。ローカル PC 内のコンテナに仮想ブラウザを 起ち上げ、画面転送技術により、インターネットをセキュアに閲覧することができます。 ローカル PC とコンテナのリソースは、論理的に分離されているため、相互に参照できない仕組みになっています。ローカルPC 内にコンテナを作成する仕様のため、サーバー上でコンテナを作成する RevoWorks SCVX よりもサーバー台数を大幅に減らすことができます。

RevoWorks SCVX と RevoWorks Browser 用途における違い

  •  インターネット接続系とLGWAN 接続系の間の通信 を 画面転送で 行う必要がある場合は、 RevoWorks SCVX を選択してください。 業務端末ではなくサーバー上の仮想ブラウザがインターネットに接続するため、 非常に高いセキュリティ を実現します。
  •  業務アプリケーションによっては、特定のブラウザでなければ正常に動作しない（もしくは動作を保証しない）ものもあります。 Firefox だけでなく Chromeを利用したい場合は、RevoWorks Browser を選択してください。
  •  Web 会議を使う業務が多い場合は、 RevoWorks Browser を選択してください。

RevoWorks SCVXとRevoWorks Browserのハイブリッド構成

RevoWorks SCVX は、 2016 年以降、 100 団体を超える 自治体で採用・導入されたインターネット分離のソリューションですが、利用できるブラウザが Firefox に限定され、 Web 会議システムに対応していないなどの制限があります。一方、 RevoWorks Browser は、先進的なローカルコンテナ技術の採用だけでなく、時代のニーズに合った機能を提供しています。

この2つの製品の管理サーバー （RevoWorks サーバー） は統合されており、ユーザーやポリシーは共通して管理／設定することができるようになっています。また、RevoWorks SCVX と RevoWorks Browser の同時アクセスライセンスは、合算したライセンス分で両製品を利用できるようになっています。

例えば、LGWAN 接続系からインターネットを閲覧する場合は、画面転送型の RevoWorks SCVX を利用し、インターネット接続系からセキュアなインターネット閲覧や Web 会議を行う場合は、ローカルコンテナ技術の RevoWorks Browser を利用するというように使い分けることができます。
